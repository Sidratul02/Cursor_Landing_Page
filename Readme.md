# Cursor landing page clone

## Project Overview

- Desktop-first recreation of the Cursor developer tool landing page.
- Built using HTML and CSS only.
- Focuses on layout structure, spacing, typography, and color accuracy.
- No JavaScript, animations, or interactive elements used.
- Designed specifically for large screens (desktop only).

## Sections Recreated

- Top Navigation Bar
- Hero Section with product preview
- Trusted By / Company Logos
- Feature Sections (two-column layouts)
- Feature Cards Grid
- Testimonials
- Use Cases / Stories
- Changelog / Updates
- Team / About Section
- Final Call-to-Action
- Footer with multi-column links
